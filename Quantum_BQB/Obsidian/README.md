## 🚀 옵시디언에서 보는 방법 (3단계)

### 1단계: 플러그인 설치 (1분)

옵시디언 설정 화면에서:

```
1. 옵시디언 열기
2. 왼쪽 하단 ⚙️ (설정) 아이콘 클릭
3. 왼쪽 메뉴에서 "커뮤니티 플러그인" 선택
4. "제한 모드 끄기" 버튼 클릭 (노란색 경고가 뜨면 확인)
5. "찾아보기" 버튼 클릭
6. 검색창에 "HTML Reader" 입력
7. "Phibr0" 또는 "nuthrash"가 만든 플러그인 찾기
8. "설치" 버튼 클릭
9. "활성화" 토글 켜기 (파란색으로 변경)
```

💡 **팁**: 설치 후 바로 활성화되지 않으면 옵시디언을 재시작하세요!

---

### 2단계: HTML 파일 넣기 (1분)

#### 📥 옵시디언 폴더 찾는 법

**방법 1 (쉬움):**
```
옵시디언에서:
설정(⚙️) → Files & Links → 맨 위에 "Vault folder" 경로 확인
→ 그 경로 옆 📁 폴더 아이콘 클릭하면 탐색기/Finder 열림!
```

**방법 2 (직접 찾기):**
- Windows: `내 문서\옵시디언 vault 이름\`
- Mac: `문서/옵시디언 vault 이름/`

#### 📋 파일 넣기

1. 이 GitHub에서 HTML 파일 4개 다운로드 (Code → Download ZIP)
2. ZIP 압축 풀기
3. HTML 파일 4개를 **옵시디언 폴더에 복사-붙여넣기**
   - Windows: `Ctrl + C` → `Ctrl + V`
   - Mac: `Cmd + C` → `Cmd + V`

✅ **완료!** 옵시디언에 파일이 나타날 거예요.

---

### 3단계: 파일 열기 (5초)

```
1. 옵시디언 왼쪽 사이드바에서 HTML 파일 찾기
   (예: bqb-beginner.html)

2. 파일 클릭!

3. 끝! 바로 렌더링되어 보입니다.
```

📌 **안 보이면?**
- 파일 우클릭 → "Open with HTML Reader" 선택
- 또는 옵시디언 재시작

#### 💻 Git 사용자라면?

터미널에서 간단하게:

```bash
# 옵시디언 vault 폴더로 이동
cd ~/Documents/MyVault

# 저장소 클론
git clone https://github.com/[사용자명]/[저장소명].git

# HTML 파일만 필요하면
cp [저장소명]/*.html .
```
