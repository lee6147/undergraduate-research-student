BQB 플랫폼 고급 가이드

설계 규칙·회로 아키텍처·열설계·KPI — 엔지니어링 레퍼런스

22nm FD-SOI ²⁸Si / ³¹P Donor TRL 3→5 5개년 KPI
📋 프로젝트 개요

BQB 과제는 반도체 공정으로 256개 큐비트 배열(16×16)을 제작하는 3년 연구입니다. 목표는 단순히 큐비트를 만드는 것이 아니라, 대규모 배열을 동시에 제작할 수 있는 플랫폼을 구축하는 것입니다.

🎯 핵심 목표
• 큐비트 위치 정밀도: ±2nm 이내
• 균일도(uniformity): 95% 이상
• 결맞음 시간(T₂): 1ms 이상
• 배열 규모: 16×16 = 256개
• 기술 성숙도(TRL): 3 → 5

대상 독자: 반도체 공정/회로 설계 경험이 있고, TCAD 시뮬레이션·PDK 설계 규칙·크라이오 패키징·RF 회로에 대한 배경지식을 보유한 엔지니어·연구자.

⚡ 에러 버짓 (Error Budget)
대규모 배열 확장의 실질적 병목: 물리적 사이트 → 운용 → 얽힘 전환율

에러 버짓은 각 공정 단계에서 허용되는 오차를 정량적으로 배분하는 것입니다. 총 오차는 각 오차의 제곱합으로 누적되므로, 모든 단계를 동시에 관리해야 합니다.

전환 구조: N_phys → N_op → N_link
• N_phys (물리적 배열): 위치 오차, 비활성 사이트, 채널링 tail
• N_op (운용 가능): outlier, 전하 불안정, T₂ 미달
• N_link (얽힘 연결): 제어 비동기, 커플링 불균일
오차 배분 예시 (±2nm 목표)
이온 주입 산란
σ_xy ≤ 1.0 nm, σ_z ≤ 0.8 nm
STI 정렬 오차
overlay ≤ 0.5 nm
패키징 정렬
칩-패키지 정렬 ≤ 1.5 nm
총 오차 (RSS)
√(1.0² + 0.8² + 0.5² + 1.5²) ≈ 2.0 nm

🛡️ Triple-Wall QBQ 상세 설계
STI moat + T3 실드웰 + BQB-B 매립 장벽의 3중 격리
Wall-1: STI 격리 moat
PDK 구현
STI 기반 isolation moat(타일 경계) 및 keep-out 구조
설계 노브
moat 폭/간격, 타일 pitch, 금속/비아 keep-out
KPI 기여
타일 간 기판 커플링/크로스토크 감소, overlay 기준 경계 확보
Wall-2: T3 Triple Well + Guard Ring
PDK 구현
T3(True Triple Well) isolation으로 back-gate/웰 도메인 분리 + quiet guard ring + 전용 리턴/실드
설계 노브
V_BG_i(타일 back-gate), V_well_shield, guard ring bias(V_SSQ), 라인 RC 필터/차폐
KPI 기여
|∇E| 및 1/f 전하잡음 민감도 저감, detune/tune-in 안정, 77K DD-IC 근접 노이즈 내성 향상
Wall-3: BQB-B 매립 장벽
PDK 구현
BQB-B(매립 장벽/템플릿 층)로 확산 차단 및 깊이 창 정의, implant window self-alignment
설계 노브
장벽 깊이/두께, 개구 CD/형상, 저열예산 활성화 파형
KPI 기여
채널링 tail/outlier 억제, σ_xy/σ_z 공정창 확대, BEOL/금속 열예산 안전성 강화
💎 DD-IC 설계
77K 극저온 Direct-Drive IC 아키텍처

DD-IC(Direct-Drive IC)는 77K에서 작동하는 극저온 제어 회로입니다. 각 큐비트에 직접 연결되어 개별 제어 신호를 생성하며, PVT(Process-Voltage-Temperature) 변동에 강건하게 설계됩니다.

핵심 회로 블록
LO-PLL
기준 클럭 25-200MHz. 위상잡음 ≤-100 dBc/Hz @1MHz offset. 온도 변동에 대한 주파수 안정성 확보가 핵심.
DCO (Digital Controlled Oscillator)
디지털 제어 발진기. I/Q 업컨버전으로 ESR 대역(28-42 GHz) 커버. DRAG/가우시안 파형 지원.
이벤트 시퀀서
DD 시퀀스(CPMG/XY) 패턴을 메모리에서 읽어 라이브러리 실행. 조건부 분기 가능.
온칩 TDC
채널 간 스큐를 피코초 정밀도로 측정하고 Delay-trim으로 실시간 보정.
Broadcast+선택 구동
공유 펄스 버스로 전체 배열에 DD 시퀀스 동시 분배. BQB의 detune/tune-in으로 선택적 구동. O(√N) 스케일링 달성.
성능 목표 (5차년도)
• 채널 수: 256-class
• 스큐 (RMS): ≤5ps
• 전력: ≤5mW/ch, ≤0.8W/chip
• 자동 보정 루프로 온도 드리프트 보상
🌡️ 열설계 (Thermal Management)
77K 환경에서 극저온 큐비트 보호

77K 극저온 환경에서 열관리는 필수입니다. 큐비트는 밀리켈빈 수준으로 냉각되지만, DD-IC는 77K에서 작동하며 수 와트의 열을 발생시킵니다.

열관리 전략
열전도 경로 설계
DD-IC 열이 큐비트로 전달되지 않도록 차단. 본딩 와이어, 패키징 재료, 냉각 핑거 최적화.
DD-IC 전력 예산
256채널 × 5mW/ch = 1.28W. 77K 냉각기의 냉각 용량(~10W) 범위 내 관리.
열사이클 테스트
300K ↔ 77K 반복 사이클에서 웨이퍼 본딩, 패키징 접합부 안정성 검증.
⚠ 주의사항
열팽창 계수 차이로 인한 기계적 스트레스가 큐비트 위치 정밀도를 악화시킬 수 있습니다. CTE-matched 재료 선정과 플렉시블 인터커넥트 설계가 필수입니다.
🔄 폐루프 공정 상세
Generate → Verify → Correct 반복 수렴

폐루프(Closed-Loop) 공정은 제조→계측→피드백의 반복 사이클입니다. 한 번에 완벽하게 만드는 것이 아니라, 점진적으로 정밀도를 높여갑니다.

3단계 사이클
⚒️ 생성 (Generate)
BQB 개구가 정의한 좌표에 단일 ³¹P⁺ 이온을 주입. RTA(rapid thermal anneal) 또는 레이저 어닐로 열예산을 엄격히 제한.
🔬 검증 (Verify)
nano-CT/XRD-CT로 3D 도너 위치를 비파괴 매핑. APT/SIMS/TEM 교차검증으로 조성과 tail 깊이 확인. 저온 전기·광학 맵 취득.
🔧 보정 (Correct)
주입 에너지·각도·선량, 개구 형상, 패시베이션 조건, 어닐 파형 갱신. 77K DD-IC 게이트 전압 보정 테이블 업데이트.
반복 수렴 전략
• 1차 회전: 공정창 준수율 70-80%
• 3차 회전: 공정창 준수율 90-95%
• 5차 회전: 공정창 준수율 98-99%
머신러닝 모델로 보정 알고리즘을 최적화하여 수렴 속도를 높입니다.
⚠️ 위험관리
8대 기술 리스크와 대응 전략
리스크 #1: 위치 정밀도 미달
대응: BQB 재설계 + 다중 계측 교차검증. nano-CT/APT/SIMS 3중 확인으로 신뢰도 향상.
리스크 #2: 동일성(uniformity) 부족
대응: 전하트랩 저감 공정. H₂ 어닐, 계면 패시베이션으로 D_it, Q_f 억제.
리스크 #3: T3+Triple-Wall 셀 리스크
대응: test vehicle A/B 비교. Wall-2 유무에 따른 전하잡음 차이를 정량화.
리스크 #4: 패키징 정렬 오차
대응: 얼라인 마크 + X-ray 피드백. 웨이퍼-패키지 정렬을 실시간 보정.
리스크 #5: 웨이퍼 본딩 리스크
대응: 본딩 test vehicle + 열사이클 평가. 300K ↔ 77K 반복으로 기계적 안정성 검증.
리스크 #6: DD-IC 성능 미달
대응: PVT sweep + 자동 보정. 온칩 센서로 온도/전압 변동 감지 후 실시간 파라미터 조정.
리스크 #7: 얽힘 데모 지연
대응: 커플링 메커니즘 다중 옵션. 정전기 커플링, 교환 상호작용, 공진기 매개 얽힘 병렬 추진.
리스크 #8: 예산 변동
대응: 필수 산출물 최소 세트 정의. 256개 배열이 불가능하면 64개로 축소하되 공정창 KPI는 유지.
